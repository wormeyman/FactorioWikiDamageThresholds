# Plastic Bar Productivity Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Plastic bar productivity research to `factorio_productivity.py`, covering Chemical plant, Biochamber, and Cryogenic plant, and generate the corresponding wiki article.

**Architecture:** Add three new machines to existing `MACHINES` and `_MACHINE_ICONS` dicts, add a `plastic_bar` entry to `RESEARCHES`, run the script to generate the wiki output file, and update `README.md`. No new functions needed - the existing `build_columns`, `find_breakpoints`, `print_wiki_table`, and `print_text` functions handle all cases generically.

**Tech Stack:** Python 3, MediaWiki markup, git/GitHub CLI
