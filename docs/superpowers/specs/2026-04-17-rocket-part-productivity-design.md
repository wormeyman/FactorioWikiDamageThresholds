## Summary

Add Rocket Part Productivity research to `factorio_productivity.py`, generate the corresponding wiki article, and update `README.md` per maintenance rules.

| Parameter | Value |
|-----------|-------|
| Machine | Rocket silo |
| Base productivity | 0% |
| Module slots | 4 |
| Bonus per level | 10% |
| Max level | 30 |
| Cap | 300% |
