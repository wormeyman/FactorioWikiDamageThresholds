# Plastic Bar Productivity Research - Design Spec

## Overview

Add `plastic_bar` productivity research to `factorio_productivity.py`, covering all three machines that can craft Plastic bar: Chemical plant (base game), Biochamber (Space Age), and Cryogenic plant (Space Age). Generate `WikiArticles/PlasticBarProductivityResearch.txt` and update README.

## Research Parameters

- **Key:** `plastic_bar`
- **Name:** Plastic Bar Productivity
- **Tech name:** `Plastic bar productivity (research)`
- **Bonus per level:** 10%
- **Max level:** 30
- **Cap:** 300%
- **Cost formula:** `round(1000 × 1.5^N)` packs per level N, cumulative totals displayed in table. Implemented via `_cumulative_costs(1000, 30)` - same as all other productivity researches.
- **Research packs:** Automation + Logistic + Chemical + Production + Agricultural
- **Notable levels:** [10, 20]

## Machines

| Machine | Base Prod | Module Slots | DLC |
|---|---|---|---|
| Chemical plant | 0% | 3 | Base game |
| Biochamber | 50% | 4 | Space Age |
| Cryogenic plant | 0% | 8 | Space Age |

Column order in table: Chemical plant | Biochamber | Cryogenic plant (base game first, then Space Age).

## Module Configs

Same 4 configs as all existing researches:
- No modules
- Prod module 3
- Legendary Prod module 2
- Legendary Prod module 3

## Table Structure

Single wide table: 12 data columns (3 machines × 4 configs) + Level + Cumulative cost = 14 columns total.
